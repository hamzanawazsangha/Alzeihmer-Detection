Project Overview

NeuroScan AI is a deep learning-based system designed to classify Alzheimer's disease from MRI scans into four distinct categories with exceptional accuracy. The system leverages state-of-the-art computer vision techniques and transfer learning to achieve medical-grade performance.

Technical Architecture

Model Details

  • Base Model: EfficientNetB0 pre-trained on ImageNet with transfer learning
  • Fine-tuning: Layers from block5a_expand_activation onward were fine-tuned
  • Custom Layers: Two dense layers (128 units each) with Batch Normalization and Dropout (0.5)
  • Output Layer: 4-class softmax classifier for Alzheimer's stage classification
  • Input Shape: 224×224×3 (EfficientNet standard input size)

Training Strategy

  • Optimizer: Adam optimizer with learning rate 0.001 and default parameters
  • Callbacks: Early Stopping (patience=10) + ReduceLROnPlateau (factor=0.5, patience=5)
  • Regularization: L2 regularization (0.001) + Dropout layers (0.5)
  • Preprocessing: EfficientNet-specific image preprocessing with normalization
  • Data Augmentation: Random rotations, flips, and brightness adjustments

Implementation Details

Technology Stack

  • Backend Framework: Flask (Python web framework) with RESTful API design
  • Machine Learning: TensorFlow 2.x / Keras with EfficientNetB0 architecture
  • Frontend Technologies: HTML5, CSS3, Vanilla JavaScript with modern ES6+ features
  • Image Processing: OpenCV for advanced image operations, PIL for basic manipulations
  • Data Handling: NumPy for numerical computations, Pandas for data analysis
  • Deployment Ready: WSGI-compatible with support for Docker containerization
